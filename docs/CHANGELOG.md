## [0.1.0] - 2024-01-15

### Added
- 初始化项目结构
- 实现无限画布 + 语义缩放（4级视图模式：mini/summary/preview/full）
- 文件卡片拖拽功能 + 对齐线
- 数据流转可视化（带箭头的贝塞尔曲线连线）
- AI 对话面板（可调整宽度）
- 动态字体大小（14px / scale * 80%）
- 支持 CSV/Excel/JSON 文件类型展示
- GitHub Actions CI/CD 配置
- 自动版本管理和 PR 工作流
