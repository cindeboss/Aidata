# DataClean AI

AI 原生数据清洗桌面工具 - MVP 阶段 UI Demo

## 项目简介

DataClean AI 是一款类似 PowerBI 的 AI 原生数据清洗工具。核心理念是协作式 AI 数据处理：

- AI 缺乏业务上下文，需要通过对话获取
- 左右分栏：左侧展示数据，右侧 AI 对话
- AI 主导对话，渐进式完成清洗
- 全局记忆：跨项目学习业务知识

## 当前状态

MVP 1 阶段 - 数据导入 + AI 对话 UI Demo

## 功能特性

- 无限画布 + 语义缩放
- 文件卡片拖拽 + 对齐线
- 数据流转可视化
- AI 对话面板

## 技术栈

- 前端：Electron + Vue 3 + TypeScript + Vite（计划）
- 后端：Python + FastAPI（计划）
- 当前：纯 HTML/CSS/JS Demo
